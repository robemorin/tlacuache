Actividad: Seguimiento de Trayectoria con Control Proporcional vs. Compensado

Objetivo: Implementar un script en Python que simule el seguimiento de trayectorias complejas por parte de una partícula, comparando dos estrategias de control: Control Proporcional Puro (P) vs. Control Proporcional + Compensación de Velocidad de Referencia (P+Feedforward) usando series de Fourier para parametrizar las trayectorias.

Descripción del Problema:

Una partícula debe seguir (rastrear) una trayectoria deseada compleja en 2D. Se compararán dos enfoques de control usando las mismas ganancias para evaluar cómo la compensación de velocidad mejora el seguimiento. La trayectoria será una figura geométrica conocida (lemniscata, epitrocoide, etc.) parametrizada en función del tiempo.

Instrucciones:

  • Parametrizar una trayectoria deseada como función del tiempo: xd(t), yd(t) preferentemente usando transformadas de Fourier o fractales para mayor complejidad.
  • Implementar dos controladores con ganancias idénticas:
    • Controlador 1 (P): u = -Kp * (x - xd)
    • Controlador 2 (P+FF): u = ẋd - Kp * (x - xd) (donde ẋd es la velocidad deseada)
  • Usar el mismo integrador numérico (Euler o RK4) para ambos controladores con tolerancias estrictas y verificación de estabilidad numérica.
  • Graficar ambas trayectorias en la misma gráfica con distintos colores/estilos.
  • OBLIGATORIO: Mostrar en las leyendas los valores de ganancia (Kp) usados en ambos casos con notación científica si es necesario.
  • Calcular y visualizar el error de seguimiento para ambos controladores.
Estrategias de Control a Comparar:
Aspecto Control P (Proporcional Puro) Control P+FF (Con Compensación)
Ley de Control u = -Kp * e u = ẋd - Kp * e
Ganancia Kp Idéntica (ej: 1.5) Idéntica (ej: 1.5)
Feedforward No Sí (velocidad deseada)
Error esperado Mayor retraso Menor retraso

Trayectorias a Seguir (Elige al Menos Una)

Opción 1: Lemniscata de Bernoulli

Parametrización:

x(t) = (a * cos(t)) / (1 + sin²(t))
y(t) = (a * cos(t) * sin(t)) / (1 + sin²(t))

Donde a = 2 es el parámetro de escala y t ∈ [0, 2π].

Opción 2: Epitrocoide

Parametrización:

x(t) = (R + r) * cos(t) - d * cos(((R + r)/r) * t)
y(t) = (R + r) * sin(t) - d * sin(((R + r)/r) * t)

Parámetros: R = 3, r = 1, d = 2 (circulo rodante).

Opción 3: Figura de Lissajous

Parametrización:

x(t) = A * sin(a * t + φ)
y(t) = B * sin(b * t)

Parámetros recomendados: A = 3, B = 3, a = 3, b = 2, φ = π/2.

Opción 4: Espiral Logarítmica

Parametrización:

x(t) = a * exp(b * t) * cos(t)
y(t) = a * exp(b * t) * sin(t)

Parámetros: a = 0.1, b = 0.1 (crecimiento suave).

Requisito de Control:
El sistema dinámico es: ẋ = u
Error de seguimiento: e = x - xd
Métrica de desempeño: RMSE = √(1/N * Σ(e²)) para toda la trayectoria.
El controlador P+FF debe tener error significativamente menor que P puro.

Métodos Numéricos (Implementar desde Cero)

Método de Euler Explícito

x[n+1] = x[n] + dt * f(x[n], u[n])

Ventaja: Simple. Desventaja: Menos preciso.

Runge-Kutta Orden 4 (RK4)

k1 = f(x[n], u[n])
k2 = f(x[n] + dt/2 * k1, u[n])
k3 = f(x[n] + dt/2 * k2, u[n])
k4 = f(x[n] + dt * k3, u[n])
x[n+1] = x[n] + dt/6 * (k1 + 2*k2 + 2*k3 + k4)

Ventaja: Mayor precisión. Desventaja: Más cálculos.

Parámetros Técnicos

  • Ganancia Proporcional: Kp = 1.5 (o valor a elegir, pero igual para ambos controladores)
  • Paso de Tiempo: dt = 0.001 s (recomendado para mayor precisión)
  • Tiempo Total: T = 2π s (al menos una oscilación completa de la trayectoria)
  • Número de Puntos: N = T / dt ≈ 6283 puntos
  • Velocidad Deseada: ẋd = d(xd)/dt (derivada de la trayectoria)
