¿Qué herramientas o aplicaciones se utilizan para el trabajo individual?

>

Poner el puntero o hacer click sobre la image para desplegar la información. Para abrir la herramienta de click en el nombre de mismo en su descripción.

Raiz2pi

Proyecto personal que contiene materiales autorevisables de practica para matemáticas, cuyo repositorio es libre en GitHub

GeoGebra

Calculadora gráfica para visualizar funciones, geometría, álgebra y cálculo. Ideal para experimentar con conceptos matemáticos complejos.

WolframAlpha

Motor de conocimiento computacional que resuelve ecuaciones diferenciales, integrales y problemas de física. Esencial para verificar resultados en ingeniería.

MATLAB & Simulink

Software estándar en ingeniería para análisis numérico, simulación de sistemas de control y procesamiento de señales. Indispensable en mecatrónica.

Overleaf (LaTeX)

Editor de LaTeX colaborativo en línea, usado para escribir reportes, tesis y artículos científicos con formato matemático profesional.

Kahoot!

Plataforma de aprendizaje basada en juegos. Útil para repasos rápidos de conceptos clave de física o matemáticas de forma interactiva.

Socrative

Herramienta de evaluación que permite a los profesores obtener retroalimentación instantánea del entendimiento de los alumnos sobre un tema.

ExamDojo

Plataforma para crear y practicar con exámenes. Ayuda a prepararse individualmente para evaluaciones, identificando áreas de oportunidad.

¿Qué herramientas o aplicaciones se utilizan para el trabajo colaborativo?

>

Poner el puntero o hacer click sobre la image para desplegar la información

Canvas LMS

Plataforma de gestión del aprendizaje (LMS) que centraliza el curso. Facilita la entrega de proyectos en grupo, foros de discusión y la comunicación oficial del equipo.

Microsoft Teams

Hub central de comunicación. Integra chat, videollamadas y almacenamiento de archivos para mantener al equipo sincronizado y organizado durante todo el proyecto.

GitHub

Repositorio de código y control de versiones. Esencial para que el equipo de software colabore en el código de Arduino o Python, revise cambios y gestione tareas.

Desmos

Su función "Desmos Classroom" permite a un equipo resolver y visualizar problemas matemáticos en tiempo real, facilitando la colaboración en la fase de análisis.

Google Colab

Cuadernos de Python compartidos y ejecutables en la nube. Ideal para que el equipo desarrolle y pruebe algoritmos de control o análisis de datos de forma conjunta.

CircuitLab

Simulador de circuitos en navegador. Permite al equipo de electrónica diseñar y probar esquemáticos de forma colaborativa, compartiendo los diseños con un simple enlace.

Tinkercad

Plataforma sencilla para diseño 3D y simulación de circuitos. Varios usuarios pueden ser invitados a un proyecto para co-diseñar piezas o probar circuitos de Arduino.

Octave

Potente software para cálculo numérico. Usado en equipo con GitHub, permite compartir y versionar scripts para resolver sistemas de ecuaciones o modelar sistemas.

¿Utilizas las herramientas para trabajo individual y colaborativo?
¿Qué potencial encuentras en una u otra?

>

¿Utilizas las herramientas para trabajo individual y colaborativo?

Sí, siempre y cuando se cumplan ciertas condiciones como

  • Sea accesible para el alumnado y para mi.
  • Que el uso de herramienta esté justificado.
  • Su uso favorezca más el aprendizaje en comparación si no se usa.

¿Qué potencial encuentras en una u otra?

Cada una tiene su ventaja, personalmente uso primero actividades colaborativas con el propósito de que los alumnos se apoyen entre sí (andamiaja pedagógico), y luego actividades individuales para que cada uno pueda demostrar lo que aprendió y reforzar su aprendizaje.

De esta manera les doy la pauta a los alumnos que se preparen para aprender y practicar un tema, y por otro el alumno se siente obligado a aprender ya que está consiente que al final será evauado individualmente.
